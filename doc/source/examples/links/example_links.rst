**********************************
H2: Download Links
**********************************
You can provide links to files in the local filesystem for download

H3: Download from current folder
================================
Download a source file from the same folder as the reStructuredText document:
:download:`download example_links.rst <example_links.rst>`

H3: Download from another folder
================================
Download a source file from a different folder in the document tree
:download:`download ../includes/example_includes.rst <../includes/example_includes.rst>`

H3: Download from an included file
==================================
Download a source file from a reStructuredText document that's included in another reStructured Text file
Note: The link to the downloaded file must be relative to the reStructuredText document that includes the document with the link

Download the included file:
:download:`download ../includes/rst_files/include_example_subsections.rst <../includes/rst_files/include_example_subsections.rst>`

**********************************
H2: External Hyperlinks
**********************************
External hyperlinks can be made by reference, such as:
- Providing a direct link: https://docutils.sourceforge.io/docs/user/rst/quickref.html#hyperlink-targets
- Linking by reference to a `Hyperlink Target Documentation`_ defined in the text document
- Linking by plain text inline syntax `Hyperlink Target Documentation in Plaintext <https://docutils.sourceforge.io/docs/user/rst/quickref.html#hyperlink-targets>`_.

.. _`Hyperlink Target Documentation`: https://docutils.sourceforge.io/docs/user/rst/quickref.html#hyperlink-targets
